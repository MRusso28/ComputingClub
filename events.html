Spring 2019 Hackathon

Compete head to head against other students in a one day coding competition

Learn More

Date & Time: April 23rd, 10am-7pm, CCE-030
The Hackathon is an all day coding challenge where you have all day to program something surrounding a theme revealed the day of competition. Teams can consist of 1-4 students. Teams create presentations that show off what they made so that judges can pick the top 3 teams, all of which win cash prizes. Teams can create websites, mobile apps, or desktop apps.

Learn the basics as well as advanced Github tips and tricks

Learn more

Date & Time: March 11th, 7-8pm, CCE-030
Github and Git can be intimidating to learn. Luckily for us, we have a guest speaker who will be sharing with us his knowledge of the technology.
